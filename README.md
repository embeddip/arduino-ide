# EmbedDIP for Arduino IDE

This repository packages **EmbedDIP** as an Arduino-IDE library.

## Supported Targets

- ESP32 boards using the Arduino-ESP32 core
- ESP32 boards with PSRAM enabled are recommended for camera and larger image workloads

The default test target used by the example workspace is:

```text
esp32:esp32:esp32:PSRAM=enabled
```
